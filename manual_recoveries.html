Help - Manual Recoveries
About Manual Recoveries:

Manual recoveries are deductions that are subtracted from an employee's gross income before tax calculation. These recoveries include items such as quarters rent, electricity charges, advance repayments, etc.

How to use this page:
  1. Add a Recovery:

    Enter the Regiment Number, select the recovery type, enter the amount, and add any relevant description.

  2. Editing and Deleting:

    Use the Edit and Delete buttons in the table to modify or remove existing recoveries.

  3. Search:

    Use the search box to filter entries by regiment number.

  4. Export:

    Export the current list of recoveries to Excel for record-keeping or further analysis.

Available Recovery Types:
  • Leave Recovery: Recoveries related to leave payments
  • RHA Recovery: Running Horse Allowance recoveries
  • HRA Recovery: House Rent Allowance recoveries
  • Audit Recovery: Recoveries identified during audit
  • Recovery From Other BN: Recoveries transferred from other battalions
Bulk Upload:

You can upload multiple recoveries at once using an Excel file. The file should have the following columns: Regiment No, Name, Rank, Recovery Type, Amount, Description. Ensure that the Recovery Type matches one of the available options from the dropdown.
